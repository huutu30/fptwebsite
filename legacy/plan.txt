IMPORTANT

Vấn đề lớn nhất: Web bạn là SPA (React + Vite) → Google crawler khó đọc nội dung. fpt.vn dùng SSR (Server-Side Rendering). Tuy nhiên, Google đã hỗ trợ crawl JavaScript khá tốt, nên giải pháp react-helmet-async + sitemap + structured data vẫn khả thi cho SPA.

Quyết định quan trọng cần bạn xác nhận:

Bạn muốn bắt đầu từ Phase nào? Phase 1 (SEO technical) là nền tảng bắt buộc. Phase 2 (trang gói cước) mang lại SEO impact lớn nhất.
Domain chính thức của web bạn là gì? Cần để set canonical URL và sitemap.
Bạn có kế hoạch chuyển sang SSR (Next.js) không? Nếu có, chiến lược sẽ khác. Nếu giữ Vite SPA thì react-helmet-async là đủ.
Open Questions
WARNING

SPA vs SSR: Web bạn là Vite SPA, trong khi fpt.vn dùng SSR. Google đã hỗ trợ crawl JS nhưng SSR vẫn tốt hơn cho SEO. Bạn có muốn migrate sang Next.js trong tương lai không? Nếu không, giải pháp react-helmet-async + pre-rendering vẫn khả thi.
